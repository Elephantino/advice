how it works?

( n + 3 )

  • n = 0 => ( 0 + 3 ) = 3-й элемент
  • n = 1 => ( 1 + 3 ) = 4-й элемент
  • n = 2 => ( 2 + 3 ) = 5-й элемент
  • n = 3 => ( 3 + 3 ) = 6-й элемент
  • n = 4 => ( 4 + 3 ) = 7-й элемент

( -n + 3 )

  • n = -0 => ( -0 + 3 ) = 3-й элемент
  • n = -1 => ( -1 + 3 ) = 2-й элемент
  • n = -2 => ( -2 + 3 ) = 1-й элемент
  • n = -3 => ( -3 + 3 ) = 0-й элемент
  • n = -4 => ( -4 + 3 ) = -1-й элемент

( 4n - 1 )

  • n = 0 => ( 4 * 0 - 1 ) = -1-й элемент
  • n = 1 => ( 4 * 1 - 1 ) = 3-й элемент
  • n = 2 => ( 4 * 2 - 1 ) = 7-й элемент
  • n = 3 => ( 4 * 3 - 1 ) = 11-й элемент
  • n = 4 => ( 4 * 4 - 1 ) = 15-й элемент
